# Gryffs AutoHunter
Automate those hunting clicks! This is meant to be used on the individual hunting page, I'm not sure yet if it'll work on the groups page...

## How to Use
1. Go to the hunting page and select the first Gryff from the list.
2. Open the browser's console with either Ctrl + Shift + J (Chrome/Edge) or Ctrl + Shift + K (Firefox). You can also right click anywhere on the page and click 'Inspect' and then click 'Console' at the top of the developer window.
3. Copy the code from main.js above and paste it into the console.
4. Press enter and sit back!

Don't use the tab while it's going; the script will automatically stop at the end or if it can't find the button, so you don't need to worry about it clicking on random things.
